# Tramacastilla disturbance palaeoecology

This repository contains all information that belongs to the paper Julián-Posada et al. (in prep)

It includes drivers and ecological consequences of natural and human-induced disturbances in Tramacastilla lacustrine sequence (1682 m a.s.l., Southern Central Pyrenees), obtained through a multi-proxy approach, that includes fossil pollen and spores, sedaDNA and macrocharcoal.

File "tramacastilla\_disturbance\_palaeoecology\_core\_data.Rmd" and pdf includes all code needed to obtain data to perform statistical analyses included in the paper.
File "tramacastilla\_disturbance\_palaeoecology\_analyses.Rmd" and pdf includes the code needed to perform all the analyses included in the paper.
File "tramacastilla\_disturbance\_palaeoecology\_figures.Rmd" includes all code needed to obtain the figures of the paper.
